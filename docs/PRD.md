# JC-App PRD

## 目的

JC-App は、青年会議所の年度ごとの組織運営を支える Web アプリです。会員、役職、委員会、予定、出欠、資料、通知をひとつの場所で管理し、年度切り替え時の引き継ぎ作業を軽くすることを目的にします。

## 想定利用者

- 理事長、専務理事、事務局
- 委員長、副委員長、幹事
- 一般会員
- 将来的に複数 LOM の管理者

## Ver.1 の対象範囲

### 1. ログイン

- Supabase Auth を利用する想定
- メールアドレスとパスワードによるログイン
- 将来的には招待制、パスワード再設定を追加

### 2. ホーム

- 現在年度と所属 LOM を表示
- 直近の予定
- 自分の出欠未回答
- 新着資料
- 通知

### 3. 会員管理

- 会員の基本情報を管理
- 氏名、連絡先、入会年度、ステータスなどは年度をまたいで保持
- 年度ごとの役職、委員会、権限は別管理

### 4. 年度管理

- LOM ごとに年度を作成
- 現在年度を設定
- 前年度の役職、委員会、権限、所属情報をコピーして新年度を作成
- コピー後に新年度の体制を編集

### 5. スケジュール管理

- 例会、理事会、委員会、事業、締切などを登録
- 対象年度と LOM に紐づけ
- 出欠対象イベントとして利用可能

### 6. 出欠管理

- イベントごとに出席、欠席、未定、未回答を管理
- 管理者は回答状況を一覧で確認
- 会員は自分の回答を変更

### 7. 資料共有

- イベントや年度に紐づく資料を登録
- Supabase Storage を利用する想定
- 閲覧権限は年度ごとの権限に従う

### 8. 通知・リマインド

- アプリ内通知を管理
- 出欠未回答、予定前日、資料追加などを通知対象にする
- メールやプッシュ通知は将来拡張

### 9. 管理画面

- LOM 設定
- 年度設定
- 権限管理
- 会員招待
- 基本マスタ管理

## 対象外

- 議案承認フロー
- AI 機能
- AI 議事録
- 会費決済
- QR 受付

## 成功条件

- iPhone で迷わず予定、出欠、資料を確認できる
- 年度切り替え時に前年度情報をコピーできる
- 会員基本情報と年度別所属情報が分離されている
- 玉島青年会議所で使い始められ、他 LOM 展開を妨げない

## 権限の考え方

- `owner`: LOM 全体管理
- `admin`: 年度、会員、予定、資料の管理
- `committee_manager`: 所属委員会の予定や出欠確認
- `member`: 自分の情報、予定、出欠、資料閲覧

権限は年度ごとに付与します。同じ会員でも年度が変われば役割が変わるためです。
