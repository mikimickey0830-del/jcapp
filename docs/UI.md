# UI Design

## 基本方針

JC-App は iPhone で使う場面を優先します。例会や委員会の前後に、片手で予定、出欠、資料をすばやく確認できる画面にします。

## 画面幅

- 最小幅: 320px
- 主対象: iPhone 13/14/15 系の 390px 前後
- PC では中央寄せの最大幅を設け、横に広がりすぎないようにします。

## ナビゲーション

- モバイルでは下部ナビゲーションを基本にします。
- Ver.1 の主要導線は「ホーム」「予定」「資料」「会員」「管理」です。
- 管理機能は権限があるユーザーだけに表示します。

## 見た目

- 青年会議所らしい信頼感を出すため、白、濃紺、明るい青を中心にします。
- ただし画面全体を青一色にせず、状態表示には緑、赤、黄も使います。
- カードは小さめの角丸にし、情報を読みやすく並べます。

## ログイン画面

- LOM 名を大きく表示します。
- メールアドレス、パスワード、ログインボタンを配置します。
- Supabase Auth 接続前でも画面の形が確認できるようにします。

## ホーム画面

ホームには以下を表示します。

- 現在年度
- 自分の役職、委員会
- 直近の予定
- 出欠未回答
- 新着資料
- 通知

## アクセシビリティ

- タップ領域は 44px 以上を目安にします。
- 文字サイズは小さくしすぎません。
- 状態を色だけで表さず、ラベルも併記します。

## レスポンシブ方針

- 主要コンテンツは `max-width: 430px` のモバイルアプリ風レイアウトを基準にします。
- タブレット以上では余白を増やして中央に配置します。
- ボタンや入力欄は横幅いっぱいを基本にします。
